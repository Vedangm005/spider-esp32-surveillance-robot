# 🕷️ S.P.I.D.E.R.
### Smart Patrolling Intelligent Detection Exploration Robot

S.P.I.D.E.R. is an intelligent quadruped surveillance robot built using ESP32, designed for real-time monitoring, object detection, and remote patrolling in hazardous or restricted environments.

---

## 🚀 Features

- 🤖 Quadruped robot with precise servo-based locomotion  
- 📷 Real-time video streaming using ESP32-CAM  
- 🧠 Intelligent object detection using YOLOv8  
- 📡 Wireless dashboard for remote monitoring and control  
- 🚨 Live alerts for detected objects and threats  
- ⚡ Efficient multitasking using Embedded C/C++  

---

## 🛠️ Tech Stack

### Hardware
- ESP32 (Main Controller)
- ESP32-CAM (Video Streaming)
- PCA9685 16-Channel PWM Driver
- Servo Motors (Leg Movement)

### Software
- Embedded C/C++
- YOLOv8 (Object Detection)
- Web Dashboard (Monitoring & Control)

---

## 🧩 System Architecture

- ESP32 handles locomotion, communication, and coordination  
- ESP32-CAM streams live video feed over WiFi  
- YOLOv8 processes frames for object detection  
- PCA9685 controls all servos independently via I2C  
- Web dashboard provides live feed, alerts, and manual control
